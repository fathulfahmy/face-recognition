## Getting Started

### Prerequisites

- Python >= 3.13

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/fathulfahmy/face-recognition.git
   ```
2. Install dependencies
   ```sh
   pip install .
   ```
